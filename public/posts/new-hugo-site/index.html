Hugo is now driving this site

Jan 2, 2019 in WEBSITE
hugo dev curtpoff.com
1 min read

Having spent the past couple months wrestling with Jekyll and Ruby gems - or more specifically: gem dependancies - I sought a new static site generator.

After perusing StaticGen a few times and trying out various test deploys on Netlify using their handy tools, Hugo came out the winner. It’s written with Go and the templating seemed straightforward and I can still produce alll of my updates using Markdown. Everything else basically came down to editing the config file. I got my tips from the Hyde-Hyde theme how-to.

I still have my curtpoff.com homepage and resume living elsewhere. But GitHub Pages is no longer involved.
